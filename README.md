# API Aprendendo na Comunidade

Projeto de API REST desenvolvido com Spring Boot e PostgreSQL.

## Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Funcionalidades
- CRUD de participantes
- CRUD de cursos
- CRUD de matrículas
- Relacionamentos JPA

## Relacionamentos
- OneToMany
- ManyToOne
- ManyToMany

## Banco de Dados
PostgreSQL

## Endpoints principais
- /participantes
- /cursos
- /matriculas
